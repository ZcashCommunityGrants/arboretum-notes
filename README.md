# Zcash Arboretum Notes

This repository holds meeting notes, summaries, and agendas for the Zcash Arboretum working group. It is intended solely for documenting discussions and decisions related to Arboretum calls.

## Contributing
- To submit meeting notes or agenda items, please open an issue in this repository.
- **Do not** post grant proposals or other unrelated content here.

## Grant Proposals
For grant proposals, please use the Zcash Community Grants repository:
https://github.com/ZcashCommunityGrants/grants

## Repository Structure
- `AllArboristCallNotes/`: Detailed retrospectives and summaries of Arboretum meetings.
- `AgendaArchive/`: Archived agendas of past calls.

Questions or concerns? Open an issue in this repository or reach out on the Zcash Community Forum.
