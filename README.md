# Arboretum Meeting Notes

This repository collects summaries and detailed notes from Zcash Community Grants (ZCG) Arboretum calls. It serves as a historical record of discussions, decisions, and technical updates from the Zcash ecosystem.

## Structure
- AllArboristCallNotes/
  - Arborist Call XX-Notes.md — Detailed notes from each bi‐weekly Arboretum call.
  - Sandblasting Retrospective - Summary.md — Postmortem summary of the June 2022 sandblasting incident.

## How to Browse
1. Clone the repo:
   ```bash
   git clone https://github.com/ZcashCommunityGrants/arboretum-notes.git
   cd arboretum-notes
   ```
2. Open the `AllArboristCallNotes/` directory to find individual meeting notes by date and topic.
3. Use your editor or viewer (Markdown preview) to read summaries and full call transcripts.

## Contributing
- To add new meeting notes, create a new Markdown file under `AllArboristCallNotes/` following the naming convention `Arborist Call NN-Notes.md`.
- For retrospective or special-topic summaries, use descriptive filenames such as `Sandblasting Retrospective - Summary.md`.
- Submit a pull request and include the meeting date and list of attendees in your header.
